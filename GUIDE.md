# Руководство: помощник по таблицам (Google Таблицы)

Простыми словами, без технических деталей. Это помощник, который работает с
вашими **Google Таблицами**: находит и читает данные, создаёт таблицы и вкладки,
вписывает и дописывает значения, ищет и заменяет текст, оформляет ячейки. Вы
пишете обычными словами, что нужно, — он делает.

> Важно: изменения в таблицах помощник показывает заранее — вы подтверждаете.

## Что можно попросить

| Вы говорите | Что делает помощник |
| --- | --- |
| «Найди таблицу „Бюджет“» | Ищет ваши таблицы по названию. |
| «Покажи данные из первого листа» | Читает и показывает значения ячеек. |
| «Создай таблицу „Учёт расходов“ с вкладкой „Январь“» | Создаёт таблицу и добавляет вкладку. |
| «Добавь эту строку в конец таблицы» | Дописывает новую строку в конец данных. |
| «Впиши эти суммы в столбец B» | Записывает значения в нужные ячейки. |
| «Найди в таблице строку с Ивановым» | Ищет текст, ничего не меняя. |
| «Замени везде „шт.“ на „штук“» | Находит и заменяет текст по таблице. |
| «Сделай шапку жирной и залей серым» | Оформляет ячейки (шрифт, цвет, заливка). |
| «Очисти данные в столбце C» | Убирает значения, оставляя оформление. |

Можно комбинировать: «Создай таблицу „Клиенты“, впиши эти строки и сделай первую
строку жирной шапкой».

### Находим и смотрим таблицы

| Инструмент | Что делает | Пример запроса |
| --- | --- | --- |
| `sheets_list` | Показывает список ваших таблиц (можно по части названия). | «Найди таблицу „Бюджет“» |
| `sheets_get_info` | Показывает название таблицы и список её листов/вкладок. | «Какие вкладки есть в этой таблице?» |
| `sheets_read_range` | Читает значения из ячеек (диапазон, например A1:C10). | «Покажи данные из первого листа» |
| `sheets_find` | Ищет текст в таблице, ничего не меняя. | «Найди в таблице строку с Ивановым» |
| `sheets_get_formatting` | Показывает оформление ячеек (цвет, шрифт, формат чисел). | «Какого цвета залита эта ячейка?» |

### Создаём и наполняем

| Инструмент | Что делает | Пример запроса |
| --- | --- | --- |
| `sheets_create` | Создаёт новую таблицу. | «Создай таблицу „Учёт расходов“» |
| `sheets_add_tab` | Добавляет в таблицу новую вкладку/лист. | «Добавь вкладку „Январь“» |
